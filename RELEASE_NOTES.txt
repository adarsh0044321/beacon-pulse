📡 Beacon & Pulse v1.0.3 - Release Notes
=======================================

Beacon & Pulse is a low-latency, hardware-accelerated remote desktop screen sharing and control system over local networks built entirely in Rust.

What's New in v1.0.3
-------------------

🚀 New Sharing Modes (Multi-Target Grid & Split Screen)
- Display Sharing Mode: Share an entire monitor instead of just a single window.
- Multi-Window Grid Mode: Select multiple windows to share simultaneously, composited in a clean grid.
- Dual-Window Side-by-Side: Select exactly 2 windows to share side-by-side in split-screen format.

⌨️ Keyboard & Input Robustness Upgrades
- Win32 SendInput Layout Independence: Key events are now injected using hardware scan codes (KEYEVENTF_SCANCODE), resolving input lag and preventing lock status LEDs (Caps Lock, NumLock) from flashing.
- Extended Key Flag Support: Added proper handling for Arrow keys, Page Up/Down, Home, End, Delete, Insert, NumPad Enter, Right Ctrl, and Right Alt.
- Automatic Stuck Keys Release: Implemented an active KeyboardCleanupGuard that automatically releases all held keys (like Shift, Ctrl, Alt, Win) when the client disconnects or connection drops.
- Loopback Feedback Isolation: Tagged injected keystrokes with a unique signature (0xBEAC0D) to ignore loopback feedback loops when testing locally on the same PC.

🛠️ Core Stability & Self-Healing Capture
- H.264 Software Fallback: If hardware encoding (NVENC/AMF/QSV) fails or driver device loss occurs, it dynamically falls back to OpenH264 software encoding.
- D3D11 Video Processor Caching: Caches blitters and video processors in the WGC backend, reducing allocation overhead and driver crashes under high resolutions (2K/4K).
- Watchdog Recovery: Robust background monitoring service with exponential back-off and registry-driven launch parameters.
- Standalone Installers: Self-extracting setup wizards (BeaconSetup.exe and PulseSetup.exe) that automatically configure firewall rules.
